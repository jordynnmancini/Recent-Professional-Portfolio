Intro

Hello there! My name is Jordyn, and I am an aspiring Full-Stack Web Developer in Columbus, OH. I am currently a student in the Ohio State Coding Boot Camp, and will be graduating in July 2021. I've learned a lot so far - particularly using Javascript. I have experience in creating and managing RESTful API's using Node.js, mySql/Sequelize, and Mongo DB. I also have experience writing front-end applications using third-party API's. This portfolio of mine showcases some of my favorite projects I've worked on so far using both front-end heavy language and back-end heavy language. Enjoy!

Client-Side Projects

Weather Dashboard | This app uses front-end Javascript, CSS/Bootstrap, and HTML to create a mobile-responsive UI. It pulls data from the OpenWeather API and allows the user to search for a city's weather. It uses Local Storage to append searched cities to the screen so the user can view the weather dashboard of past searches.

  • GitHub Repo
  • Deployed Application

Watchaholic - a Movie Search App | This was a group project, and is built with HTML, CSS, Bootstrap, Javascript, JQuery. It interacts with 3 Third-Party API Sources - WATCHMODE, OMDB and New York Times. This allows the user to search for movies and tv shows, and then aggregates important information pulled from those 3 APIs. It's main purpose is to show all streaming services that contain this title along with a direct link to them.

  • GitHub Repo
  • Deployed Application

Javascript Coding Quiz | This was one of the first applications I made after learning Javascript. It has very simple HTML, and mainly uses Javascript to append items to the DOM. It prompts the user to start the coding quiz which triggers a timer; when the user answers incorrectly, 10 seconds are deducted from the time. They get points for each question they answer correctly, and the final score is determined based on their points and seconds left. It then uses Local Storage to save and append high scores.

  • GitHub Repo
  • Deployed Application

Server-Side Projects

TiddeR Blog Forum | This is a group project that uses the MVC approach to build and connect the server-side and client-side. We used node.js and the express npm package to write the server-side code, and Handlebars for the front-end. We used mySql for the database, and used the sequelize npm package to interact with it.

  • GitHub Repo
  • Deployed Application

The Tech Blog | This was a solo project using MVC to build a tech blog site. It allows users to signup, login, view their dashboard with blogs they have written (along with comments from other users), post more blogs, view blog posts from other users, and leave comments. The information is stored in a mySql database.

  • GitHub Repo
  • Deployed Application

Employee Tracker | The user can access this Content Management System via a command-line application. It uses node.js and the inquirer npm package to prompt the user with questions, and allows them to view and modify the employee data from the mySql database.

  • GitHub Repo
  • Recorded Demo
